Privacité / Privacidad
Cette application collecte anonymement le nombre d'utilisateurs et les réponses soumises dans le jeu. Vous pouvez demander la suppression et l'exclusion de ces données dans le menu des options du jeu.
Esta aplicación recopila de forma anónima el número de usuarios y las respuestas enviadas en el juego. Se puede solicitar la eliminación y exclusión de estos datos en el menú de opciones del juego.
